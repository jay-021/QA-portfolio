API Testing on Formspree

A detailed overview of the API testing I performed on the Formspree API to ensure the reliability of my contact form.

Functionality Testing

  • Tested valid form submissions to confirm successful processing and email delivery.
  • Verified error handling for invalid data and server unavailability.
Postman

Security Testing

  • Investigated Formspree's input validation to protect against XSS and SQL injection.
  • Checked for rate limiting mechanisms to prevent abuse.

Performance Testing

  • Measured the time taken for the form to submit and for the success/error message to appear.
  • Conducted basic load testing by submitting the form multiple times in quick succession to assess how it handles concurrent requests.
